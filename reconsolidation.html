Cognitive Science · Field Notes · Vol. IV

Every recall reopens the write

Volume III confessed a gap: real memories, unlike recordings, are reshaped by every act of recall. This volume closes it. Below, a memory of a finished event, one the world can no longer correct. Retell it and watch the details wash out. Ask one leading question and watch something that never happened move in for good.

An interactive essay · follows Vol. III, "Cut open" Read time · 7 min

Psychology found this out three times, by three different roads. Frederic Bartlett (1932) had people retell a folktale at widening intervals and watched each retelling drift further toward the teller's own conventions: war canoes became boats, seal hunts became fishing trips, and the strange parts quietly conformed or vanished. Elizabeth Loftus showed that the drift can be steered: ask an eyewitness "how fast was the car going when it passed the barn?" and a barn that was never on the film starts appearing in their memory of it. And the neuroscience of reconsolidation (Nader and colleagues, 2000) supplied the mechanism: a reactivated memory becomes temporarily labile and must be written down again, together with whatever else is present at the moment of recall. Three findings, one shape. Recall is not read-only.

None of that fits the engine of Volume II as built, because that engine always had the world on tap to retrain against. So this volume changes the model in exactly one place and nowhere else:

  • The event is over. The world signal is gone for good. What remains is a cue, the mere prompt "tell me about it," and a readout trained, while the event was still happening, to rebuild the whole experience from the cue-driven ripples.
  • The only writer left is recall. Between recalls the readout is untouched, and an untouched memory here keeps perfectly, because the past does not drift. Consolidation now runs only during remembering, and the only training signal it can find is the retelling itself, plus whatever the situation slips into it.
  • Nothing can veto the story. In Volume II every distortion washed out in seconds because the world was still streaming in. Here there is no world to appeal to. Whatever a retelling writes, stays written, until the next retelling writes over it.

§ 01The model, changed in one place

The reservoir below is the same 90-unit network as Volume II: fixed sparse random weights, leaky nonlinear units, one linear readout. The event it experienced was a waveform with a distinctive detail riding on a plain fundamental. During the event, consolidation trained the readout to rebuild the full waveform from the cue alone. Now the event is over. The black line is the event as it actually happened (we, the experimenters, kept a copy; the network has no access to it). The red line is the event as the network currently remembers it, reconstructed live from the cue-driven reservoir state.

Press Retell it and the memory is recalled with consolidation on: the readout retrains toward a slightly smoothed, more conventional copy of its own last retelling. That is reconsolidation, and the smoothing is Bartlett's schema pull. Press Ask a leading question and the same thing happens, except the retelling now carries a suggestion mixed into it, a component that was never in the event. The network has no way to file it separately.

The meters are scored against the original event. Gist is the fundamental, the broad shape of what happened. Detail is the distinctive component that made this event this event. Intrusion measures how much of the suggested, never-experienced component the current memory contains. The network sees none of these numbers. From the inside, every reconstruction is equally fluent.

§ 02What to look for

  1. Leave it alone. Do nothing. Watch the fidelity strip stay flat, for minutes, forever. An unrecalled memory in this model does not fade, because nothing is training and the past does not drift. Notice what that makes it: a frozen readout is exactly Volume III's recording. Between recalls, a memory is a recording. It only becomes something else, for better and worse, at the moment you open it.
  2. Retell it, a few times. Each retelling smooths the waveform a step toward the generic. The detail meter falls roughly a third per retelling; gist barely moves; fidelity steps down and stays down. This is Bartlett's serial reproduction, run on weights instead of undergraduates. Gist survives because the cue keeps it anchored. Detail conforms because the schema pull is the only editor present. Nothing is erased; the memory is re-fit, each time, to the previous telling.
  3. Ask a leading question. One press. The intrusion meter jumps, and a component that was never in the event is now confidently part of the reconstruction. Now retell it innocently a few times: the intrusion stays, schematized at the same gentle rate as the genuine details. The misinformation effect, live. The implanted detail is stored by the same reconsolidation that stores real ones, so later retellings carry it forward as if it had always been there. The system has no tag that distinguishes experienced from suggested. Neither, the evidence says, do we.
  4. Relive the event. The only undo. Re-experiencing the original restores detail and flushes the intrusion completely. And that is the punchline: for a real past event there is no relive button. Volume II's distortions healed in seconds because the world was still there to retrain against. A finished event has no world left. Whatever recall writes is, from then on, the event.

§ 03The loop, re-plumbed

Volume II's load-bearing claim was that the loop is what makes it memory. That loop has not been cut here. It has been re-plumbed. Draw it and look at where the return path runs now:

For a live situation, the cycle closed through the world, and the world pushed back: act on a wrong memory and reality corrects you. For a finished event, the cycle closes through the self's own reconstructions. The stream that consolidation learns from is the retelling; the retelling is whatever the readout produced last time, gated and framed by the self, in whatever company the recall happens. The engine's defining virtue in Volume II, that training never stops, is untouched. It has simply lost its anchor. The same mechanism that kept memory current in a drifting world now lets a well-placed question write fiction into it.

Reconsolidation and suggestibility are not a feature and a bug. They are one mechanism, seen on a good day and a bad one.

And notice what the model cannot feel. The fidelity meters exist because we kept the original waveform offstage. The network has no such oracle, and neither do you. From the inside, reconstruction is just as fluent when it is wrong: the flashbulb-memory studies that followed people for ten years after 9/11 found accuracy falling for the first year while confidence never budged, and errors, once introduced, repeated in report after report. A reconstructive system fails silently to its owner. Volume III said that silent failure was the mark of a recording; the uncomfortable amendment is that on this one axis, a rememberer is no better off, because the error signal that drives correction needs a world to compare against, and the world of a past event is gone.

§ 04What this is and isn't

The experiments above are real behaviors of a real (tiny) echo state network under reconsolidation-style training. The usual honest caveats:

First, the schema pull is asserted. Each retelling here trains toward a measured blend of the previous telling and the cue's plain shape. Real schematization is not a fixed blending coefficient; it emerges from a lifetime of knowledge about how stories usually go, which is why Bartlett's participants drifted toward their culture's conventions, not toward a sine wave. The direction of drift is the model's claim; the mechanism producing that direction is borrowed, not earned.

Second, nothing decays here, on purpose. Real memories fade unrehearsed. The model deliberately removes every other source of change, so that whatever you see on the fidelity strip is attributable to recall and recall alone. It is the same ablation logic as Volume III: hold everything constant, permit one arrow, and attribute the difference to the arrow.

Third, the consolidation rule had to grow up. Volume II's readout was trained by a rule that chases: it stayed accurate only because it never stopped correcting, which is fine for a memory that is never off duty. A memory that must hold still between recalls needs a rule that actually converges, so consolidation here runs the recursive form of least squares. Biology faces the same problem, a trace that must survive long gaps in training, and solves it with offline replay during sleep rather than with better algebra.

Fourth, implanting is too easy here. One question, one wiggle, written in at full strength. Human memory has brakes this model lacks: reconsolidation only opens under particular reactivation conditions, implantation studies succeed in a minority of participants, and critics of the false-memory literature (Pezdek among them) rightly note that a mechanism existing is not proof of how often it fires. The model shows the door exists and opens inward. How often it gets pushed is an empirical question about people, not weights.
